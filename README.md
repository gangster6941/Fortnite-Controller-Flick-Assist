**Snap to enemy** with analog stick flick movement.

---

## 🎯 Features

- Left/right flick bind  
- Trigger to engage aim  
- Adjustable flick distance  
- Works with PS/Xbox controllers  
- Safe input layer

---

## 🛠️ Use

1. Connect controller  
2. Run `Loader.exe`  
3. Toggle flick mode with D-pad
